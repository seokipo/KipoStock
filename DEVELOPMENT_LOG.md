## [V1.3.8] - 2026-03-12
### 🚀 주요 변경 사항
- **당일매매일지 API 직접 연동 (100% 무결성 확보)**: 종가 베팅 탐색 대상을 구성할 때, 복잡하고 불완전할 수 있는 내부 로컬 로깅(json, 매매 이력 등)에 의존하지 않고, HTS 화면과 동일한 **KIS 당일매매일지 API(ka10170)**를 직접 호출하여 오늘 "단 한 주라도 매수한 종목" 리스트를 가장 정확하게 가져오도록 대폭 수정했습니다.
- 이를 통해 네트워크 문제나 중간 에러로 로컬 기록에 남지 않았던 종목들까지 완벽하게 잡아내어, KipoStock 필터의 사각지대를 완전히 없앴습니다. (자기 요청 반영 ❤️)
