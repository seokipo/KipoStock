# KipoStock AI 명령어 매핑 리스트 (v6.1.21) 🤖📚

자기야! AI 비서가 알아들을 수 있는 자연어 표현과 대응되는 실제 명령어 매핑 리스트야. ❤️

## 1. 계좌 및 리포트 (Account & Reports)

| 자연어 의도 (Voice/Text) | 대응 명령어 (`ACTION:`) | 설명 |
| :--- | :--- | :--- |
| "오늘 수익 얼마야?", "잔고 보여줘", "리포트 뽑아줘" | `report` | 당일 매매 종합 리포트 출력 |
| "오늘 뭐 샀어?", "매매 일지 보여줘" | `today` | 시간순 매매 내역 리스트 |
| "손실 많은 것부터 보여줘" | `today son-` | 손실 금액 기준 내림차순 정렬 조회 |
| "전략별로 수익 어때?" | `today jun` | 매수 전략별 정렬 조회 |

## 2. 매매 제어 (Trading Control)

| 자연어 의도 (Voice/Text) | 대응 명령어 (`ACTION:`) | 설명 |
| :--- | :--- | :--- |
| "매매 시작해", "감시 가동해", "시동 걸어" | `start` | 실시간 감시 엔진 시작 |
| "매매 멈춰", "잠깐 중단해" | `stop` | 실시간 감시 엔진 정지 |
| "1번부터 자동 시작해", "3번부터 오토 돌려" | `auto [번호]` | 지정된 프로필부터 시퀀스 가동 |
| "전부 꺼", "자동 시퀀스 중단해" | `auto 0` | 자동 시퀀스 및 감시 중지 |

## 3. 설정 및 환경 (Settings & UI) 🚀 [NEW]

| :--- | :--- | :--- |
