# 구현 계획서 - 안드로이드 앱 전환

KipoStock 애플리케이션을 갤럭시 노트 22 울트라에 최적화된 고품질 안드로이드 인터페이스로 전환합니다.

## 컨셉: "모바일 커맨드 센터"

현재 주식 매매 API(키움 등)는 윈도우 전용 32비트 환경이 필수적이므로, 모든 로직을 안드로이드에 직접 이식하는 것은 불가능합니다.

**해결책**:
1.  **PC 웹 서버 (백엔드)**: 기존 파이썬 코드가 PC에서 매매 로직과 API를 계속 수행합니다.
2.  **모바일 앱 (프론트엔드)**: 노튼 22 울트라에서 실행되는 아름답고 현대적인 웹앱(PWA)입니다. 내 PC와 보안 연결되어 로그 모니터링, 설정 변경, 시작/중단 명령을 처리합니다.

## UI/UX 디자인 (전략)
- **OLED 최적화 다크 모드**: 깊은 블랙 배경을 사용하여 배터리 효율과 시각적 몰입감을 높입니다.
- **네온 액센트 컬러**: 수익률과 활성 상태를 강조하기 위해 선명한 형광색을 사용합니다.
- **글래스모피즘 (Glassmorphism)**: 세련되고 투명한 느낌의 UI 요소 배치.
- **노트 22 울트라 최적화**: 대화면과 S펜 조작을 고려한 시원시원한 레이아웃.

## 주요 변경 사항

### 1. 백엔드 통합
- `Kipo_GUI_main.py`에 경량 웹 서버(FastAPI 등)를 추가하여 모바일 통신 채널 확보.
- 제공 API:
    - `GET /status`: 현재 실시간 로그 및 매매 상태 조회.
    - `POST /command`: 시작, 중지, 리포트 요청 전송.
    - `POST /settings`: 모바일에서 즉시 매매 파라미터 업데이트.

### 2. 모바일 대시보드
- HTML5/CSS3/Vanilla JS를 이용한 최신 대시보드 제작.
- WebSocket을 이용한 실시간 로그 스트리밍 (폰에서 PC 로그가 흐르듯 보임).

## 검증 계획
1.  모바일 대시보드 목업(Mockup) 제작 및 검토.
2.  로컬 네트워크를 통한 폰-PC 간 통신 확인.
3.  명령 전송 시의 지연 시간 및 UI 반응성 검증.
