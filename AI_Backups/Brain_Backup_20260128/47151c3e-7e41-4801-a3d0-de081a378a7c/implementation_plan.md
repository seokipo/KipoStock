# 구현 계획서 - KipoBuy 안드로이드 앱 변환

KipoBuy 주식 자동매매 시스템을 안드로이드 기기에 최적화된 전문적인 모바일 웹 애플리케이션(PWA)으로 변환합니다. 모듈형 레이아웃을 채택하며, 요청하신 대로 알람 기능을 제외합니다.

## 제안된 변경 사항

### [확정] 모바일 엔진 고도화
- **알람 기능 완전 제거**: `mobile_server.py` 및 프론트엔드 UI에서 알람 및 소리 관련 UI와 로직을 0%로 유지. (오직 매매 제어와 모니터링에만 집중)
- **외부 접속(Ngrok)**: 집 밖(LTE/외부망)에서도 스마트폰으로 접속 가능하도록 Ngrok 터널링 환경 구축.

### [확정] 안드로이드 네이티브 패키징 (.apk)
- **갤럭시 노트 22 울트라 최적화**: WebView를 사용하여 폰 화면 전체를 매매 환경으로 활용.
- **유연한 주소 설정**: 내부 IP(`192.168.0.5`)와 외부 Ngrok 주소를 선택적으로 적용 가능하게 구현.

## 검증 계획

1. **기능 이식**: 데스크탑 GUI의 핵심 기능(매수, 매도, 리포트, 설정)이 모바일에서 정상 동작하는지 확인.
2. **원격 제어**: Ngrok 주소를 통해 외부망에서 앱이 정상적으로 서버에 연결되는지 확인.
3. **알람 부재**: 어떤 상황에서도 모바일 기기에서 소리나 알람이 발생하지 않는지 확인.
