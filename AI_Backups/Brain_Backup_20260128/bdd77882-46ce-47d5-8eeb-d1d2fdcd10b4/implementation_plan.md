# 임플리멘테이션 플랜 - V5.4.10: 손익 정렬 및 입력값 자동 보정

본 계획은 매매 리포트 조회 시 손익 순으로 정렬하는 기능과, 사용자가 익절/손절 값을 입력할 때 부호를 자동으로 보정해주는 안전장치를 추가하는 것을 목표로 합니다.

## 제안된 변경 사항

### 1. `chat_command.py` (명령어 기능 확장)
- **`today` 명령어 개선**: `today son` 옵션을 추가하여 매매 리포트를 손익금(또는 수익률) 기준으로 정렬하는 로직 구현. (기본 내림차순, `-` 붙이면 오름차순)
- **`help` 명령어 업데이트**: 새로 추가된 `today son` 옵션에 대한 설명을 도움말에 추가.

### 2. `Kipo_GUI_main.py` (UI 및 데이터 검증 강화)
- **입력값 자동 보정 (`save_settings`)**:
    - **익절 값**: 입력된 값이 음수일 경우 절대값(양수)으로 자동 변환.
    - **손절 값**: 입력된 값이 양수일 경우 음수(-)로 자동 변환.
    - 보정된 값은 즉시 UI 입력 필드에 반영되어 사용자에게 피드백 제공.
- **버전 정보 업데이트**: 윈도우 타이틀을 **V5.4.10**으로 변경.

### 3. `build_exe.py`
- 빌드 파일명을 **KipoStock_V5.4.10_Auto**로 변경.

- [ ] UI에서 손절값에 양수(`5`) 입력 후 저장 시 음수(`-5`)로 바뀌는지 확인.
