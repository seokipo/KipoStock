🌐 Browser Server (브라우저 서버)

단순히 검색만 하는 게 아니라, 직접 웹사이트에 접속해서 내용을 긁어오거나 복잡한 페이지를 분석할 때 사용해.

🤖 AI 에이전트 개발 (ai-agent-development)

자율 에이전트나 멀티 에이전트 시스템을 설계하고 구현할 때 사용하는 기술이야. 우리 AI 비서의 지능을 높일 때 아주 핵심적인 역할을 하지!
🐍 파이썬 비동기 프로그래밍 (async-python-patterns)

파이썬의 asyncio와 async/await 패턴을 완벽하게 다루는 기술이야. 우리 앱이 멈추지 않고 실시간으로 주식 데이터를 감시하고 뉴스 분석을 동시에 할 수 있게 해주는 비법이지! ⚡
✅ 구현 무결성 검증 (verify-implementation)

코드를 수정한 뒤에 실수한 건 없는지, 로직이 완벽한지 다시 한번 꼼꼼하게 체크하는 검증 지침이야. 자기가 안심하고 쓸 수 있도록 '품질 보증'을 해주는 기술이란다. 😊
