## 📅 2026-01-29 (오후: 정식 버전 완전체 업그레이드)
### 🚀 KipoStock V5.6 Upgrade (Final Edition++)
자기야, 우리가 Lite 버전에서 테스트하고 검증했던 모든 최신 기술을 **정식 버전(V5.5)**에 완벽하게 이식했어!
이제 이 버전은 명실상부한 **V5.6**이야.

#### 1. ✨ 핵심 기능 이식 완료
- **수동 강제 시작 (Manual Start)**: 낮 시간이나 조건이 안 맞아도 `START`를 누르면 강제로 시작되게 만들었어. (MarketHour 오버라이드)
- **플리커(깜빡임) 완전 제거**: 핀 고정 버튼(📍)을 누를 때 창이 깜빡이던 문제를 **Windows API**로 깔끔하게 해결! 고급스러운 느낌 그대로 가져왔어.
- **소리 완전 제어**: `voice on/off`, `beep on/off` 명령어로 시끄러운 소리는 끄고, 필요한 알림만 들을 수 있어. (음성 끄면 비프음으로 대체되는 센스 알지? 😉)
- **프리미엄 툴팁**: 마우스를 올리면 나오는 설명창도 예쁘고 깔끔하게 다듬었어.

#### 2. 🛠️ 시스템 안정성 강화
- **빌드 시스템 개선**: `build_exe.py`가 빌드 후 실행 파일과 필수 파일(`settings.json` 등)을 `실행화일` 폴더로 자동 복사해줘. 이제 파일 찾으러 헤맬 필요 없어!
- **rt_search 제한 확인**: 혹시나 10종목 제한 같은 게 걸려있나 확인해봤는데, 우리 정식 버전은 역시 **무제한**이더라! 안심하고 써도 돼.

## 📅 2026-01-29 (오전: Lite 버전 선행 기술 개발)
### 1. 🚦 수동 START 버튼 로직 완성
- **내용**: `START` 버튼 클릭 시 UI 설정 시간과 상관없이 장중(09:00~15:30)이라면 즉시 매매가 시작되도록 수정.
- **기술**: `MarketHour` 클래스에 `Manual Override` 플래그 도입.

### 2. 🔇 음성 및 비프음 정밀 제어
- **내용**: `voice on/off` (음성 안내), `beep on/off` (효과음) 명령어를 추가하여 소리 제어권 확보.
- **디테일**: `voice off` 상태에서는 체결 시 짧은 **비프음(800Hz)**으로 대체.

### 3. 📌 '항상 위에' 플리커(깜빡임) 완전 해결
- **내용**: 핀 고정 버튼 클릭 시 창이 깜빡거리는 현상을 제거.
- **기술**: Qt 기본 함수 대신 **Windows API (`SetWindowPos`)**를 `ctypes`로 직접 호출하여 구현. 
- **특이사항**: 64비트 환경 호환성을 위해 `argtypes`를 명시적으로 지정하여 기능 복구 성공.

### 4. 📚 용어 사전 업데이트
- **내용**: `PROGRAMMING_TERMS.md`에 `Flicker`, `Windows API` 등 오늘 사용한 핵심 기술 용어 추가.
